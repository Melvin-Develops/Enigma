# 🚀 Base-36 Alphanumeric Enigma Simulator

A modern, Python-based implementation of the historic Enigma M3/M4 architecture, expanded to support a full **Base-36 character set** (A-Z, 0-9). This project demonstrates the application of modular arithmetic and object-oriented design to adapt classical cryptography for modern alphanumeric data.

---

## ✨ Key Features
- **Extended Character Set:** Supports uppercase letters (A-Z) and numerical digits (0-9).
- **Authentic Signal Path:** Accurately simulates the plugboard, three-rotor assembly, and reflector logic.
- **State-Based Encryption:** Implements polyalphabetic substitution via a dynamic rotor stepping mechanism that advances with every keystroke.
- **Full Reciprocity:** Utilizing a reciprocal reflector allows the same machine settings to be used for both encryption and decryption.

## 🛠 Tech Stack
- **Language:** Python 3.10+
- **Mathematics:** Modular Arithmetic ($N=36$)
- **Architecture:** Object-Oriented Programming (OOP)
